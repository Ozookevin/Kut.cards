Ku.Cards browser extension is currently being developed. Feel free to use the current beta. Follow the instructions below.

 

1. Click here to download the  Kut.Cards extension beta.

 -current beta supports Google Chrome and Microsoft Edge  

 

2. Go to chrome://extensions/ and check the box for Developer mode in the top right.

 

3. Locate the ZIP file on your computer and unzip it. 

 

4. Go back to the chrome://extensions/ page and click the Load unpacked extension button and select the unzipped folder for your extension to install it.
